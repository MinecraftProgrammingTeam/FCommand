# FCommand

#### 介绍
用户原要求：mc原版中按F是交换副手，现在需要一款插件，能够在检测玩家按F的时候，屏蔽交换副手这个动作，并且执行指令 希望能够设置以玩家身份执行，以OP身份执行，以控制台执行 最好还可以使用%player%变量 版本：paper 1.18.2 如果能兼容其他更多的版本当然更好

#### 安装教程
直接拖入plugins文件夹，然后reload或重启服务器（推荐直接重启服务器）即可。
